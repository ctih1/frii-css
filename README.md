## frii-css
pronounced "freaks" (**/friːks/**)
CSS library that aims to mimic the look and feel of the old frii.site frontend

## Demo
Visit https://demo.friicss.ctih1.fi for the demo. 
Source code for the demo is in index.html

## Selling points
- Doesn't look that ugly
- Requires like no styling
- Supports modals, loaders, buttons, input, switches, tables, and dropdowns

## How to use
I recommend downloading the minified CSS and hosting it yourself as my server is kind of slow.
